# Система кластеризации данных для дата-аналитики

## Описание проекта
Информационная система для исследования методов кластеризации и построения аналитических моделей. Система позволяет загружать данные, проводить их предобработку, применять различные алгоритмы кластеризации и визуализировать результаты.

## Функциональные возможности

### 1. Загрузка данных
- Поддержка форматов CSV и Excel
- Автоматическое определение структуры данных
- Валидация загружаемых файлов

### 2. Анализ и статистика
- Базовая статистика по данным
- Анализ пропущенных значений
- Обнаружение выбросов
- Анализ типов данных

### 3. Предобработка данных
- Заполнение пропущенных значений (6 методов)
- Удаление выбросов (2 метода)
- Нормализация данных

### 4. Кластеризация данных
- **K-Means**: для сферических кластеров
- **DBSCAN**: для кластеров произвольной формы
- **Иерархическая кластеризация**
- **Гауссовы смеси (GMM)**
- **Спектральная кластеризация**

### 5. Визуализация результатов
- 2D и 3D визуализация кластеров
- Графики распределения
- Интерактивные графики
- Экспорт результатов
